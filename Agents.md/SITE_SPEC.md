# 寒舍群網站規格說明

## 1. 專案目的
本專案為「寒舍群」品牌網站的首頁初版，採用 Bootstrap 建置，先完成主要資訊架構與視覺區塊，方便後續依正式設計稿持續調整。

## 2. 技術與檔案結構
目前專案結構如下：

- `index.html`：首頁主結構與內容
- `css/custom.css`：所有自訂樣式集中管理
- `img/`：本機圖片素材，輪播圖與其他示意圖皆放在此資料夾
- `bootstrap-5.3.8-dist/`：Bootstrap 本機資源備份

## 3. 前端技術
- HTML5
- Bootstrap 5.3.x
- 自訂 CSS 分離至 `css/custom.css`
- 版面採響應式設計，支援桌機與行動裝置

## 4. 首頁區塊規格
頁面由上到下依序包含：

1. 選單列
   - 左側品牌名稱：寒舍群
   - 右側導覽連結：首頁、關於我們、學員反饋、聯絡資訊
   - 使用 Bootstrap navbar

2. 輪播圖區
   - 共 5 張輪播圖
   - 使用 Bootstrap carousel
   - 圖片來源為本機 `img/` 資料夾
   - 第一張輪播含標題、說明與按鈕遮罩層

3. 關於我們區
   - 左圖右文版型
   - 左側為示意圖片
   - 右側為品牌介紹與重點說明

4. 學員反饋區
   - 共 4 個 CARD 模組
   - 採 Bootstrap card + grid 排版
   - 後續可直接替換為真實學員回饋內容

5. 頁尾資訊區
   - 寒舍群連結
   - 公司資訊
   - 版權宣告

## 5. 樣式規範
所有自訂樣式統一維護在 `css/custom.css`，目前包含：
- 品牌色變數
- 選單背景與毛玻璃效果
- 區塊標題底線樣式
- 輪播圖尺寸與遮罩
- 關於我們圖片陰影與圓角
- 學員卡片 hover 效果
- 頁尾樣式

## 6. 圖片與路徑規則
- 輪播圖與內容圖請優先放入 `img/`
- 圖片路徑請使用相對路徑
- 若檔名含空白或特殊字元，建議使用 URL 編碼或先重新命名為英文檔名
- 目前輪播圖已對應本機圖片檔，若之後更換素材，需同步更新 `index.html`

## 7. 維護原則
- 新增或修改樣式時，優先調整 `css/custom.css`
- 若要新增頁面，建議延用同一份 `css/custom.css`，維持全站一致性
- 若要擴充輪播或卡片內容，盡量維持 Bootstrap 的 grid 與 component 結構
- 不建議把樣式再寫回 `index.html`，避免未來維護分散

## 8. 後續交接建議
交接時可優先確認以下項目：
- 圖片素材是否齊全，且檔名與程式引用一致
- 頁尾公司資訊是否要改成正式資料
- 學員回饋文字是否要替換成正式文案
- 是否需要新增內頁或導覽錨點

## 9. 備註
目前專案以首頁初版為主，尚未拆分多頁結構。若未來網站擴充，建議先規劃共用導覽與共用頁尾，再逐步建立其他頁面。
